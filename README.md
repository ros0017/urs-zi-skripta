# URS — Završni ispit (skripta)

Pregledna web-skripta s riješenim završnim ispitima iz kolegija **Ugradbeni računalni sustavi (URS)**, FER.

Pokriva ispite **2021–2024**: STM32F407 (GPIO, EXTI, SysTick, Flash, retargeting), PicoBlaze (asembler + VHDL), FPGA/CPLD/PLD teoriju i VHDL primjere — sve uz detaljna objašnjenja.

## Sadržaj
- **Teorija** — koncepti koji se ponavljaju (PROM, PAL/GAL, Spartan-3, PLL, RC oscilator, Flash WS, prefetch, boundary-scan, reset, VHDL osnove)
- **ZI 2024 / 2023 / 21·22 / 2021 / 20·21 + Dodatni** — zadaci s rješenjima i objašnjenjima

Otvori `index.html` u pregledniku ili posjeti deployani link.
